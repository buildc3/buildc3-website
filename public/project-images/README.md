# Project Images

Place project thumbnail images here.

Naming convention: `projectname.png` (lowercase, hyphens for spaces)
Example: `stepforces.png`, `my-new-project.png`

After adding an image:
1. Run `git add public/project-images/yourimage.png`
2. Commit and push — Netlify will deploy automatically
3. In the Admin panel (or DB), set `thumbnail_url` to `/project-images/yourimage.png`
